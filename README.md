# 🟡 Pac-Man

<!-- Скриншот или GIF -->
Реализация классической аркадной игры Pac-Man на C++ с использованием библиотеки SDL3.

## Описание

Игрок управляет персонажем Pac-Man, перемещаясь по лабиринту и собирая точки. Цель — собрать все точки, избегая столкновений с призраками. Специальные большие точки временно позволяют Pac-Man’у преследовать и поедать призраков.

## Технологии

| Компонент       | Технология   |
| --------------- | ------------ |
| Язык            | C++20        |
| Графика         | SDL3         |
| Система сборки  | Make         |
| Документация    | Doxygen      |
| Контроль версий | Git / GitHub |

## Требования

Для сборки проекта необходимо, чтобы в системе были установлены:

- **Make 3.24+**
- Компилятор с поддержкой **C++20**
